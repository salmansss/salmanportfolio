ONLINE VOTING SYSTEM

We Started building this project in October 2014 Online Voting is a system which is built to overcome all the hardles being faced while carrying out elections physically Firstly user has to register with the system to make himself eligible for online voting.
When user logs into the system,he will be sent secondary password on his registered mobile number which will work as authorization that he is the person who is trying to log in to the system.
Administrator account will be added by default and can perform all the admin related activities.

Technologies:
  • - Java Sevlet Page
  • - Netbean 6.5 or higher version
  • - JDK 1.6.0_04 or above
  • - Apacha Tomcat
  • - Microsoft Sql Server 2005
